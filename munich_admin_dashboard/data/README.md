# Local dashboard data

Put Instagram export/processed CSV files here when running locally.

This folder is intentionally kept almost empty for GitHub. The real export files can include account stats, captions, tokens, or private workflow data, so they should stay on your computer.

Expected local files include:

```text
content_insights_clean.csv
activity_posts_inventory.csv
zip_inventory.csv
stage1_summary.json
external_trending_tags.csv
posting_experiments.csv
```

The Facebook photo archive feature creates/uses `Muenchner_Knabenchor_Archive/` locally. That folder is also ignored by Git.
